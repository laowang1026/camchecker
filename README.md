# camchecker

Tool for Reconnaissance on a webcam (python3).

### Use

>	 python3 camchecker.py -u http://url/   
 
	usage: camchecker.py [-h] [-u URL]   
	optional arguments:  
	-h, --help  show this help message and exit  
	-u URL      URL to scan [required]
